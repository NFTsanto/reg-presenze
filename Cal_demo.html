⚙️ Configurazione Firebase necessaria

Per attivare il database cloud, segui questi passi:

  1. Vai su Firebase Console
  2. Crea un nuovo progetto (es: "presenze-dipendenti")
  3. Aggiungi un'app Web () e copia la configurazione
  4. Attiva Realtime Database in modalità test
  5. Attiva Authentication con Email/Password
  6. Sostituisci la configurazione nel codice (riga ~450) con i tuoi dati

ATTENZIONE: Attualmente l'app usa dati locali. Configura Firebase per il cloud!
